Please select a CSV file containing issue data. The file should include the following columns:
  • create_time - Create Time (Format: YYYY-MM-DD)
  • Carline - Carline
  • Power - Power Type (ICE, BEV, PHEV)
  • Specific Function - Specific Function
  • Function Domain - Function Domain
  • General Domain - General Domain
  • Issue Type - Issue Type
  • Description - Description (Chinese)
  • Description En - Description (English)
  • Brief_Issue - Brief Issue (Chinese)
  • Brief_Issue_En - Brief Issue (English)
Import Instructions:
  • The imported data will be appended to existing records
  • The system will automatically assign new global IDs
  • Imported records will be marked as admin imported
  • The current time will be used as the batch identifier
  • Blank lines and rows with format errors will be skipped
